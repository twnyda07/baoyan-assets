# 寶嚴禪寺財產管理系統

財產清冊網頁：**https://twnyda07.github.io/baoyan-assets/**

## 架構（與 zoom 預約系統同模式）
- **前端**：本 repo 的 `index.html`，GitHub Pages 靜態頁
- **後端**：Google Apps Script Web App（twnyda07@gmail.com 帳號）
  - **財產資料存於 Google 試算表**「寶嚴財產管理系統資料庫」（首次執行自動建立；含「財產清冊」＋「異動紀錄」兩張工作表），可直接在試算表查看／修改
  - **照片存於 Google Drive** 資料夾「寶嚴財產照片」（自動建立，前端已先壓縮到 900px 內）
  - 管理密碼存於 Script Properties，預設 `baoyan2026`
  - API 網址寫在 `index.html` 的 `API_URL`
  - 後端原始碼備份：`Code.gs`

## 功能
- **統計總覽**：全部財產／借出中／維修中／報廢 四張卡，點卡片即篩選
- **搜尋＋分類篩選**：搜名稱、編號、位置、保管人；分類籤（法器、音響設備、電器、家具⋯可自訂）
- **財產卡片**：照片（點擊放大）、財產編號（留空自動 BY-0001）、名稱、分類、位置、數量、狀態、保管人、備註
- **借用／歸還**：任何人免登入即可登記借用（填姓名）與歸還；借出中卡片顯示借用人與日期
- **異動紀錄**：頁面底部顯示近 30 筆（新增／修改／刪除／借出／歸還），完整紀錄在試算表（保留 500 筆）
- **管理人專區**（頁面底部登入，密碼預設 `baoyan2026`）：新增／編輯／刪除財產、上傳照片、開啟資料試算表、更改密碼

## 部署後端（首次，約 5 分鐘）
1. 用 twnyda07 帳號開 https://script.google.com → 新專案，命名「寶嚴財產管理系統後端」
2. 貼上 `Code.gs` 全部內容，儲存
3. 部署 → 新增部署作業 → 類型「網頁應用程式」→ 執行身分「我」、存取權「所有人」→ 部署（授權時允許試算表＋Drive 權限）
4. 複製 `/exec` 網址，填入 `index.html` 第一個 `<script>` 裡的 `API_URL`，push

## 修改後端
到 script.google.com（twnyda07 帳號）→ 專案「寶嚴財產管理系統後端」→ 改完程式後 **部署 → 管理部署作業 → 編輯 → 版本選「新版本」→ 部署**（沿用同一網址）。
